## 📋 **Sobre**

Uma viagem no tempo pelos computadores dos anos 90! Este projeto simula um ambiente **Windows 95** completo com efeitos CRT de TV de tubo, terminal DOS e uma área de trabalho funcional.

Desenvolvido para a **Mostra Científica 2026** do Instituto Murialdo, explorando a relação entre **tecnologia e matemática**.

---

## 🎮 **O que temos**

### 🖥️ **Sistema Retrô**
- Tela inicial com computador desligado
- Boot estilo Windows 95 (15 segundos)
- Terminal DOS automático
- Login com animação de senha
- Área de trabalho com ícones e barra de tarefas
- Menu Iniciar funcional

### 📺 **Efeitos CRT**
- Estática e ruído
- Linhas de scan
- Curvatura de tubo
- Flicker e interferências
- Ondas de distorção

### 📱 **Aplicativos**
- **Meu Computador** - Unidades de disco e propriedades
- **Windows Explorer** - Navegação de pastas
- **Gerenciador de Tarefas** - Processos e desempenho
- **Painel de Controle** - Configurações diversas
- **Bloco de Notas** - Editor de texto simples
- **Calculadora** - Operações básicas com memória
- **Sobre** - Informações do projeto

---

## 🌀 **Museu da Matemática** (Destaque)

Experiência 3D imersiva com:

- **Fractais 3D** - Mandelbrot, Julia, Sierpinski, Curva do Dragão, Floco de Koch, Árvore Fractal
- **Números Primos** - Espiral de Ulam interativa (clique nos números!)
- **Constantes** - π, e, φ, i, τ com curiosidades
- **Fibonacci** - Sequência e espiral dourada
- **Pitágoras** - Visualização interativa com sliders
- **Teoria dos Números** - Goldbach, Riemann, primos gêmeos

> 🎯 **Clique em qualquer elemento para descobrir informações!**

---

## 🚀 **Como usar**

1. Na tela inicial, **pressione qualquer tecla** para ligar
2. Aguarde o boot (15 segundos)
3. Clique em **OK** no login
4. Clique duas vezes nos ícones para abrir apps
5. Use o **Menu Iniciar** para acessar tudo
6. Clique em **⏻** para desligar
